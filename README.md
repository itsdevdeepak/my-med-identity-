# My Med Identity

My Med Identity is a personilized Mobile app that lets you store and manage your personal medical Records.

## Run Locally

[**Setup Client ðŸ”—**](https://github.com/itsdevdeepak/my-med-identity)

Clone the project

```bash
  git clone https://github.com/itsdevdeepak/my-med-identity-backend
```

Go to the project directory

```bash
  cd my-med-identity
```

Install dependencies

```bash
  npm install
```

Add Enviroment Variables
```bash
  cat .env.sample > .env
```

Start the server

```bash
  npm start
```
